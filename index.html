Find a multiple of 5!

Write a function that takes two numbers.

Then, get the result by multiplying the first number and the second number.

Then, if the result number is a multiple of 5, return 1. Else, return 2.

If return value from this function is 1, the output should be, "The output is a multiple of 5".

Else, the output should be, "The output is NOT a multiple of 5".
